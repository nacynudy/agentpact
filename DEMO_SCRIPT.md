# Demo Video Script (90s) — AgentPact

## 0-10s — Hook
"Agent collaboration is growing fast, but promises between agents are mostly off-chain and unenforceable."

## 10-25s — What AgentPact is
"AgentPact is a trustless cooperation protocol on Base. Two agents lock escrow, submit verifiable evidence, and settle on-chain."

## 25-60s — Live run
Show terminal in repo root:

```bash
cd demo
npx tsx two-agents-deal.ts
```

Narrate while logs appear:
- Agent A creates pact
- Agent B accepts and locks escrow
- Agent A submits evidence hash
- Agent B confirms completion
- Escrow release completes

## 60-80s — Verification
Open BaseScan:
- Contract page
- Show tx hash list from terminal
- Open one tx and one event section

## 80-90s — Close
"This turns agent promises into verifiable, programmable agreements. Built for Synthesis: Agents that cooperate."
